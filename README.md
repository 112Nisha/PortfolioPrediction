# PortfolioPrediction

A Flask web app for portfolio mean-variance analysis and rebalancing suggestions.

## Features
- Web portal for users to input their stock choices and weights/target return/target risk
- Computes historical mean and variance for the portfolio
- Displays the mean-variance frontier graph
- Suggests portfolio rebalancing for higher mean and/or lower variance

## Usage
2. **Run the app locally**
   ```bash
   python app.py --host <host> --port <port>
   ```
   Visit `http://127.0.0.1:8080` or <host>:<port> in your browser.
