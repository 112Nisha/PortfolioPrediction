Portfolio Optimization

Portfolio optimization is a mathematical approach to selecting the best mix of assets to maximize returns and minimize risk. The most common method is mean-variance optimization, which uses historical data to estimate expected returns and risk (variance) for each asset.

  • Mean: The average historical return of the portfolio.
  • Variance: The risk or volatility of the portfolio's returns.
  • Efficient Frontier: The set of optimal portfolios offering the highest expected return for a given level of risk.
